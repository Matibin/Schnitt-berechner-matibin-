# Notenschnitt-Berechner

Ein einfacher Notenschnitt-Berechner als Web-App – ohne Installation, ohne Abhängigkeiten.

## Nutzung

Einfach die Datei `index.html` im Browser öffnen.

## Funktionen

- Noten mit optionalem Fach und Gewichtung eintragen
- Gewichteter Durchschnitt wird sofort live berechnet
- Eingabe mit Komma oder Punkt (z. B. `2,5` oder `2.5`)
- Zeilen hinzufügen, entfernen und alles zurücksetzen
- Eingaben werden lokal im Browser gespeichert und bleiben nach dem Neuladen erhalten
- Light- und Dark-Mode (folgt der Systemeinstellung)
- Funktioniert mit jedem numerischen Notensystem (z. B. Noten 1–6 oder Punkte 0–15)
